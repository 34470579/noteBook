# 我的苹果笔记本

## 零、序言

平时积累的东西以笔记的形式分享出来

## 使用

```sh
git clone https://github.com/OMGZui/noteBook.git

# 前端
cd js
npm install
node main.js
# 浏览器
http://localhost:5000/ 查看console


# PHP
composer install
php -S localhost:8000
# 浏览器
http://localhost:8000/php/php7.php
# 命令行
cd php
php -f php7.php
或
php php7.php
```
